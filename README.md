# Snake-DQN-AI
Projeto de inteligência artificial que treina um agente para jogar Snake usando Deep Q-Learning. O objetivo é fazer a IA aprender, por tentativa e erro, a coletar comida, evitar colisões e maximizar sua pontuação dentro do ambiente do jogo.

Objetivo do projeto:
    Fase 1: Criar Snake manual / ambiente funcional
    Fase 2: Fazer a IA jogar aleatoriamente
    Fase 3: Implementar DQN simples
    Fase 4: Adicionar replay buffer
    Fase 5: Adicionar target network
    Fase 6: Treinar até pegar comida consistentemente
    Fase 7: Melhorar reward e evitar loops
    Fase 8: Testar modelo treinado com renderização
